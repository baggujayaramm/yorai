# Yorai

Yorai is a student experience connection MVP for asking students, reading lived insight, and discovering what actually works inside and across colleges. This pass also adds live-thread language, freshness signals, and context-attachment guidance while keeping events, hackathons, and collaboration as future placeholders.

## Commands

```bash
npm install
npm run dev
npm run typecheck
npm run build
```

For Prisma, copy `.env.example` to `.env`, point `DATABASE_URL` at PostgreSQL, then run:

```bash
npm run prisma:migrate
npm run prisma:seed
```
